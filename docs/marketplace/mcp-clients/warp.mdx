## Example

```json
{
  "mcpServers": {
    "@agentic/search": {
      "url": "https://gateway.agentic.so/@agentic/search/mcp"
    }
  }
}
```

See the [Warp MCP docs](https://docs.warp.dev/knowledge-and-collaboration/mcp#adding-an-mcp-server) for more details.
