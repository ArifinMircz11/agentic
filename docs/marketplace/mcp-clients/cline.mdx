## Example

```json
{
  "mcpServers": {
    "@agentic/search": {
      "url": "https://gateway.agentic.so/@agentic/search/mcp",
      "disabled": false
    }
  }
}
```

See the [Cline MCP docs](https://docs.cline.bot/mcp/connecting-to-a-remote-server) for more details.
